# Pandas data cleaning Assignment

# Objective

The objective of this assignment is to learn python basics and perform basic data exploration and also data cleaning using python pandas library.

# Task Performed

Loaded CSV dataset(Combined_dataset.csv) into a Pandas DataFrame
Explored dataset using:
head()
tail()
shape
columns
data types
Identified and removed missing values
Filtered rows and selected columns
Removed duplicate rows

Created a derived column:

total_amount = final_price × quantity (quantity column is not given in the data set so we have initialized it as 1 for all the products in the combined dataset. )

Saved cleaned dataset as a new CSV file as (cleaned_shopping.csv).
